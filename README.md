# ZiQiang-Studio-Tasks
1.通过 Python 包（package）实现对用户输入字符串的 base64 加解密，并对非法输 入做异常处理。

3.通过 Python 包（package）实现读取（用户输入的文件名）.txt 文件中的内容，将 其生成 QR Code，并本地保存为.jpg 图片文件。
